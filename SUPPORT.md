# Support

AspectIDE is an early alpha open-source project. Use GitHub Issues for reproducible bugs and focused feature requests.

## Before Opening an Issue

- Search existing issues.
- Reproduce on the latest local build or a named commit.
- Collect OS, architecture, AspectIDE version/commit, logs, and screenshots when relevant.
- Separate unrelated problems into separate issues.

## Good Support Requests

For bugs, include exact steps, expected behavior, actual behavior, and whether the problem appears in the desktop app, web dev mode, or a Rust service.

For feature requests, describe the developer workflow, the missing behavior, and where you think it belongs: Rust engine, React workbench, Tauri IPC/events, LSP/DAP, AI tools, extension system, installer, or docs.

## Not Support Channels

Do not use public issues for security vulnerabilities. Follow [SECURITY.md](SECURITY.md).
